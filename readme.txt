Plugin to change the order in which products and text appear, in the categories and on the main page.

== Description ==
When the plugin is activated in a website that uses woocommerce, it exchanges the positions of the products, above, and the text of the page, below.
This is done both with the main page and with the category pages.
The change is made automatically once the plugin is activated.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress and done


== Frequently Asked Questions ==
Do you have to configure something?
  Nothing at all, it does it automatically.
